## Requirements

```
* Swift 3.0.1
* XCode 8.3.1
* iOS 9.0 (Min SDK)
```

## Installation

GradientSlider is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:


```ruby
pod 'GradientSlider’, :git=> ‘https://github.com/nickypatson/GradientSlider’
```
